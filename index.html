Design Patterns

Design Patterns are a good way to create and solve commonly occurring problems in software. These projects demonstrate some of the design patterns used.

Linked Lists

Linked Lists are an important part of computer science as the ability to be able to point to nodes in a system is invaluable. They offer efficient ways at removing elements as one can simply delete a node and change the pointers of the nodes as opposed to shifting an entire array. Additionally, inserting can be made more efficient with linked lists as dynamic arrays can sometimes exceed the current capacity as it can reallocate and insert another copy.

Binary Search Trees

One important part of coding is data traversal. When you're dealing with a small data pool it can be easy to just wing it, however more and more data can quickly make traversal a nightmare and that is where Binary Search Trees come in. With a properly created BST, traversals are made fast and easy.

Multithreading

Multithreading allows for multiple threads to be created and run concurrently. This is important for efficiency which, in turn, leads to the maximum utilization of the computers' CPU. The files in this Git Repository are simple programs that showcase and use the methods spoken about before.
